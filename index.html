Novo Curso - Técnico de Informática para Internet

Estão abertas as inscrições para o curso Técnico de Informática para Internet.

 

O curso, com duração de 1.000 horas (18 meses), é semipresencial: 80% da carga hor ária é cumprida a distância, o que permite aos alunos economizar com deslocamentos, dedicar mais tempo ao aprendizado e flexibilizar os horários de estudo, já que pessoas inseridas no mercado de trabalho, para especialização ou transição de carreira, são as que mais procuram esse tipo de qualificação.

 

A formação tem como foco habilitar os alunos a trabalharem com softwares livres ou de código aberto e, ao fim do curso, o aluno ser á um desenvolvedor web full stack e sairá com o diploma com habilitação de técnico de informática para a internet apto a:

  • criar interfaces web, para sites ou apps;
  • prototipar aplicações on-line;
  • estruturar bancos de dados;
  • desenvolver códigos nas principais linguagens de programação web: HTML5, CCS3 e JavaScript.

Utilizar softwares livres ou de código aberto para o ensino facilita o acesso aos interessados pelo curso, uma vez que não precisarão investir em instalação de programas pagos no computador utilizado para os estudos. Outras facilidades são o valor e a forma de pagamento: 26 parcelas de R$155,00, pagos via boleto bancário.

 

Serão 2.320 vagas distribuídas nas 57 unidades do SENAI-SP que ofertarão o curso, totalizando 58 turmas, cada uma com 40 pessoas. Os alunos poderão escolher a escola mais próxima de casa ou do trabalho para cumprir a etapa presencial do curso.
